# Proxmox Helper Toolkit
A collection of scripts for Proxmox Virtual Environment

### Install scripts
```
bash -c "$(curl -fsSL -H 'Cache-Control: no-cache' https://raw.githubusercontent.com/Martan404/Proxmox-Helper-Toolkit/refs/heads/main/pht-updater?cb=$(date +%s))"
```
#### To update simply run `pht-updater`
